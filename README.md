# AI Study Helper 📚🤖

An AI-powered study assistant for Indian students (Class 6–12).

## Features
- Solve questions by typing
- Step-by-step explanations
- Simple language
- Supports Maths, Science, English

## Tech Used
- Google Gemini API
- JavaScript
- OCR (Tesseract.js)
- Vite

## Run Locally
npm install  
npm run dev

This app is for learning only. It does not promote cheating.
